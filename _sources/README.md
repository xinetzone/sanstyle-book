让编程走进生活，构建一个万物互联的社区。本社区支持：

1. 作为可运行程序的教科书，比如打造一个可交互的教学平台。
2. 作为知识分享的乐园。
3. 可讨论最前沿的技术。
4. 收录最优质的资源。
5. 其他功能。

`Sanstyle` 项目借鉴自 {term}`ExecutableBookProject`。
